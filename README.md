## WEEK 1 : Introduction

In week 1; I was introduced to the basics of HTML, mainly:

- basic HTML boilerplate 
  
  ```html
    <!DOCTYPE html>
    <html lang="en">
    <head>
    
    </head>
    <body>
        
    </body>
    </html>
  ```

- HTML Headings tags
  
  ```html
    <h1></h1>
    <h2></h2>
    <h3></h3>
    <h4></h4>
    <h5></h5>
    <h6></h6>
  ```
- Paragraph tags
  
  ```html
    <p></p>
  ```

- Images tags
  
  ```html
  <img src"images/image.png">
  ```

- anchoring tags and Linking other page
  
  ```html
    <a href="link url">Click to go to link</a>
  ```
